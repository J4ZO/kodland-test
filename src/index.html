POP IT!

¡El mejor juguete anti-estrés: Papel de burbujas infinito!

¿Qué es Pop It?

Pop It se traduce literalmente como «revienta esto». Es un juego infantil cuyo proceso puede compararse con reventar el plástico de burbujas. Pero, a diferencia de la película de burbujas tradicional, los Pop It se pueden presionar con sensaciones y sonidos característicos de forma «infinita»: decenas de burbujas de silicona simplemente se empujan hacia el otro lado.

Reglas del Juego

Los jugadores se turnan para presionar las burbujas. Es obligatorio terminar la fila que se haya comenzado. Se puede presionar como máximo 3 burbujas al mismo tiempo (deben estar una junto a la otra; no se permite presionar varias burbujas a la vez si no están pegadas). El objetivo es hacer que el oponente sea quien juegue al final. Quien hizo el último movimiento, perdió.
